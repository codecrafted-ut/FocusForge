# FocusForge 🔥

**FocusForge** is a lightweight student productivity web app built to help you stay focused without distractions.

## ✨ Features
- Clean and minimal UI
- Focus-first design (no clutter)
- Fast and lightweight
- Built for students


## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 Purpose
Most productivity apps are overloaded.  
FocusForge is different — it keeps things simple so you can actually get work done.
